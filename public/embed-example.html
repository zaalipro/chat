Chat Widget Integration Examples

This page demonstrates different ways to integrate the chat widget into your website.

1. Floating Widget (Bottom Right)

The simplest integration - adds a floating chat widget to the bottom right corner of your page.

<!-- Include the widget script --> <script src="https://yourdomain.com/embed.js"></script> <script> ChatWidget.init({ apiUrl: 'https://your-api.com/graphql', publicKey: 'your-public-key-here' }); </script>

2. Auto-Initialize with Data Attributes

Even simpler - just add data attributes to the script tag:

<script src="https://yourdomain.com/embed.js" data-api-url="https://your-api.com/graphql" data-public-key="your-public-key-here" data-position="bottom-right"> </script>

3. Inline Widget in Container

Embed the chat widget directly into a specific container on your page:

<div id="my-chat-container"></div> <script src="https://yourdomain.com/embed.js"></script> <script> ChatWidget.init({ containerId: 'my-chat-container', apiUrl: 'https://your-api.com/graphql', publicKey: 'your-public-key-here' }); </script>

4. Different Positions

You can position the floating widget in different corners:

ChatWidget.init({ apiUrl: 'https://your-api.com/graphql', publicKey: 'your-public-key-here', position: 'bottom-left' // bottom-right, bottom-left, top-right, top-left });

5. Widget API Methods

Control the widget programmatically:

// Show the widget ChatWidget.show(); // Hide the widget ChatWidget.hide(); // Update configuration ChatWidget.updateConfig({ position: 'top-left' }); // Remove widget completely ChatWidget.destroy();

6. Multiple Widgets

You can have multiple chat widgets on the same page for different purposes:

<!-- Support Chat --> <div id="support-chat"></div> <!-- Sales Chat --> <div id="sales-chat"></div> <script> // Load support chat widget ChatWidget.init({ containerId: 'support-chat', apiUrl: 'https://support-api.com/graphql', publicKey: 'support-public-key' }); // Note: For multiple widgets, you'd need to modify the script // to support multiple instances </script>
